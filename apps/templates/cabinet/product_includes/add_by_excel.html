Формат XLSX (Excel 2007-365)

Вы можете подготовить каталог товаров в программе Microsoft Excel и загрузить на наш сайт. Файл должен полностью соответствовать нашему шаблону (см. ниже). Также наш формат совместим с форматом Яндекс.Маркет Excel, за исключением того что файл нужно сохранить в формате XLSX (меню «Сохранить как...» в Excel).

Шаг 1. Скачайте шаблон

Скачать шаблон
Шаг 2. Добавьте в шаблон ваши товары

Первые четыре строки с товарами заполнены для примера. Отредактируйте их, указав свои товары

Шаг 3. Загрузите файл

Загрузите файл нажав на кнопку «Загрузить» вверху этой страницы или перетащив его в поле загрузки

    Дополнительная информация
  • Файл должен иметь расширение .xlsx, т.е. быть сохранен в формте Excel 2007-365
  • В шаблоне запрещено удалять строку с названиями полей
  • Товары должны располагаться на листе «Товары»
  • Если вы используете категории, они должны быть предварительно созданы на вкладке Категории
  • Один товар должен занимать только одну строку
  • По ссылкам на картинки должны быть доступны фото в открытом доступе
  • Для обновления товаров просто загрузите файл с id товаров, которые уже есть в базе
